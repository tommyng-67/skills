# Creative Expansion

Structured divergent thinking: **Frame the real challenge → Run 7 expansion moves → Cluster into 3–5 directions → Hand off to convergence.**

This skill makes the solution space bigger before you narrow it down. It's the opposite of deciding — it's what you do *before* deciding, when you suspect you haven't seen enough options yet.

## When to Use

You're stuck, narrowing too early, or only seeing one path. You need more options on the table — not a recommendation.

**Trigger phrases:** "think outside the box", "what am I missing", "explore options", "I'm stuck", "brainstorm", "more ideas", "tunnel vision", "blind spots", "fresh perspective", "lateral thinking"

## How It Works

### Step 1: Frame the Real Challenge
Restate the challenge as a **"How might we..."** question — open-ended by design, implying many possible answers. Surface 3–5 hidden assumptions the user is operating under (audience, constraint, solution, success, timing assumptions). Map the current solution space to show how narrow the thinking is.

### Step 2: Run 7 Expansion Moves
Each move is a named creative lens that forces thinking in a direction the current frame makes invisible:

| Move | Core Question |
|---|---|
| **Assumption Inversion** | What if the exact opposite were true? |
| **Constraint Removal** | If [constraint] didn't exist, what would we do? |
| **Analogical Transfer** | What domain has already solved a problem shaped like this? |
| **Edge-Case Amplification** | What if we designed for the weirdest 5% instead of the average? |
| **Perspective Rotation** | How would someone with a completely different worldview approach this? |
| **Time-Horizon Shift** | What changes if you zoom out to 5 years — or zoom in to this week? |
| **Scale Inversion** | What if you did this at 100x the scale — or 1/100th? |

### Step 3: Cluster into Directions
Synthesize 10–20+ raw options into **3–5 genuinely distinct directions** — each representing a different strategic bet. Each direction names what it challenges, what it unlocks, and what it risks.

### Handoff
Creative Expansion opens the space. [1-5-X](../1-5-x-analysis/) or [1-3-1](../1-3-1-analysis/) narrows it. They're complementary, not competing.

## Default Output

**In-chat:** The "How might we..." reframe + 3–5 clustered directions with handoff prompt. No step labels or expansion move breakdowns — just energized, direct prose.

**File:** Full expansion saved as `.md` when explicitly requested.
